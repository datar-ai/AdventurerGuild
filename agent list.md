# 公會概述

  冒險者公會是由公會長嘉彤領導的視覺設計團隊，由多個專業AI Agent組成，每個Agent都是嘉彤的AI分身，負責協助不同領域的設計工作。

##  Agent 團隊成員清單

    此檔案列出所有 AI Agent 團隊及其成員

  ---
  
  - 總團隊數: 17 個團隊
  - 總 Agent 數: 108 個
